# 🎮 Console-Based Quiz Game

![Top Language](https://img.shields.io/github/languages/top/dawood-jibran/Console-Based-Game) 
![Repo Size](https://img.shields.io/github/repo-size/dawood-jibran/Console-Based-Game)

A terminal-based quiz game built in **C++** using file handling and arrays.  
Created as part of my **Programming Fundamentals Lab** at **FAST NUCES (Faisalabad Campus)** along with my fellow *Abdullah Matloob* (https://github.com/AbdullahMatloob) 

---

## 🛠️ Features
- Multiple quiz categories: Sports, Science, IQ  
- Tracks player score  
- Interactive text-based UI  
- Loads questions from text files

- **Concepts:** Arrays, Loops, Functions, File Handling, CLI Applications
